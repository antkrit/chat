src.views.chat_view module¶

Chat views used to render chat template with users messages.

This module defines the following views: - chat, function that defines chat view

async src.views.chat_view.chat(request: aiohttp.web_request.Request)¶

Render chat.html template for url route /chat/<uuid> and endpoint chat.

Parameters

request – incoming request

Raises

web.HTTPNotFound – in case if uuid is wrong or chat with such uuid does not exist

Returns

rendered template chat.html until the web socket is open, web.WebSocketResponse after opening.
